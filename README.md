# Tune

A minimal instrument tuner for iOS and iPadOS.

Listens to your microphone and displays the detected pitch as a note name and frequency in real time. Supports sharps, flats, or both.

## Requirements

- iOS / iPadOS 26.2+
- Xcode with Swift 5.0

## Dependencies

- [AudioKit](https://github.com/AudioKit/AudioKit)
- [SoundpipeAudioKit](https://github.com/AudioKit/SoundpipeAudioKit)
